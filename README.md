🛠️ 사용한 기술
JavaScript: 브라우저와 Node.js 환경에서 사용되는 프로그래밍 언어
HTML5: 웹 페이지 구조 설계
CSS3: 스타일링과 레이아웃
DOM API: 문서 객체 모델을 사용한 동적 HTML 조작
